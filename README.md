# Frontend (React + Vite)

## Configuration domaine de production

Créer un fichier `.env.production` à partir de l'exemple:

```bash
cp .env.production.example .env.production
```

Puis vérifier:

```env
VITE_API_URL=https://hofatrano.tina-lalaina.site/api
```

## Lancer en local

```bash
npm install
npm run dev
```
